## **프로젝트 설명**

UCC : [https://youtu.be/V2gcpwDS10c](https://youtu.be/V2gcpwDS10c)

</br>

주요기능

- 인기 작물 추천, 월별 작물 추천
- 농작물 상세 정보 및 레시피 제공
- 농작물 가격 및 상태 정보 통계 제공
- 농작물 물주기 알림, 작물 추천 알림, 커뮤니티 활동 알림, 쪽지 메시지 알림
- 다양한 커뮤니티 (자유, 정보, 질문, 나눔)
- 인기 게시글 추천, 사진 업로드
